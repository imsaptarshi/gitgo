# gitgo

A simple GitHub API integration with GoLang

## Usage

| name            | command                                  |
| --------------- | ---------------------------------------- |
| user_info       | gitgo <user_name> info                   |
| repository_info | gitgo <user_name> repo <repository_name> |
|                 | gitgo <user_name> repo \*                |
